# 코바늘 도안 기호 체계 (Symbol System)

> 이 문서는 서술형 도안 입력 → SVG 그림 도안 변환을 위한 **기호 정의 및 입력 문법** 명세입니다.

## 1. 기본 코 (Stitch) 기호

| 입력 코드 | 한글 명칭 | 영문 약어 | 비고 |
|---|---|---|---|
| `@` | 매직링 | MR (magic ring) | 원형 도안 시작점 |
| `O` | 사슬뜨기 | ch (chain) | 평면 시작·단 시작에 사용 |
| `S` | 빼뜨기 | sl st (slip stitch) | 단 마무리·이동에 사용 |
| `X` | 짧은뜨기 | sc (single crochet) | 가장 기본 코 |
| `T` | 긴뜨기 | hdc (half double crochet) | |
| `F` | 한길긴뜨기 | dc (double crochet) | |
| `E` | 두길긴뜨기 | tr (treble crochet) | |

## 2. 변형 코 (Modifier) 기호

| 입력 코드 | 의미 | 베이스 코 | 비고 |
|---|---|---|---|
| `V` | 늘림 (한 코에 N개) | 짧은뜨기 (X) | 기본 확장수 = 2 |
| `A` | 모아뜨기 (N코를 1개로) | 짧은뜨기 (X) | 기본 확장수 = 2 |
| `blo` | 뒤이랑뜨기 (back loop only) | 다음 코에 적용 | |

> **V/A의 기본 베이스는 짧은뜨기(X)로 고정**. 다른 코 종류 기반의 늘림/줄림은 v2 이후 도입.

## 2.1 별칭 (Alias)

입력 시 정식 코드 외에 영문 표준 약어 및 단축 형태도 허용합니다. **대소문자 무관**으로 받아들이며, 내부 표현은 정식 코드로 정규화됩니다.

| 정식 코드 | 허용 별칭 | 비고 |
|---|---|---|
| `@` | `mr` | magic ring |
| `O` | `ch`, `o` | chain |
| `S` | `sl`, `_` | slip stitch (`s`는 `sc`와 혼동되므로 미사용) |
| `X` | `sc`, `x` | single crochet |
| `T` | `hdc`, `t` | half double crochet (긴뜨기) |
| `F` | `dc`, `f` | double crochet (한길긴뜨기) |
| `E` | `tr`, `e` | treble crochet |
| `V` | `inc`, `v` | increase |
| `A` | `dec`, `a` | decrease |
| `blo` | `BLO` | back loop only |

**동등 표현 예시**:

```
1: @, 6X            ≡   1: mr, 6sc        ≡   1: @, 6x
3: (1X, 1V) * 6     ≡   3: (1sc, 1inc) * 6
6: 24X, S           ≡   6: 24sc, sl       ≡   6: 24x, _
```

**파싱 규칙**: longest-match 토크나이저 사용 (예: `sl`을 `s`+`l`이 아닌 단일 토큰으로 인식).

## 3. 입력 문법 (Pattern Syntax)

### 3.1 기본 형태

```
[반복수][기호][^확장수]
```

- **반복수**: 기호를 몇 번 적용할지 (생략 시 1)
- **기호**: 코 기호 (X, F, T, E, V, A 등)
- **확장수**: V/A 기호의 확장 정도 (`^N` 형태로 표기, 생략 시 V=2, A=2)

### 3.2 표기 규칙

| 표기 | 의미 | 소비 코 → 생성 코 |
|---|---|---|
| `X` | 짧은뜨기 1회 | 1 → 1 |
| `6X` | 짧은뜨기 6회 | 6 → 6 |
| `V` | 늘림 (1코에 X 2개) = `V^2` | 1 → 2 |
| `V^3` | 1코에 X 3개 | 1 → 3 |
| `V^4` | 1코에 X 4개 | 1 → 4 |
| `A` | 2코 모아뜨기 = `A^2` | 2 → 1 |
| `A^3` | 3코 모아뜨기 | 3 → 1 |
| `2V` | 늘림(V^2)을 2회 | 2 → 4 |
| `2V^3` | V^3을 2회 | 2 → 6 |
| `2A^3` | A^3을 2회 | 6 → 2 |

### 3.3 결합 (Sequence)

여러 기호를 콤마(`,`)로 구분하여 순차 실행.

```
1X, 1V, 1X        → X V X (3코 소비, 4코 생성)
@, 6X             → 매직링 + 짧은뜨기 6회
```

### 3.4 반복 (Repetition)

소괄호 `()` + `*N`으로 패턴 반복.

```
(1X, 1V) * 3      → X V X V X V (6코 소비, 9코 생성)
(1X, 2V) * 3      → X V V X V V X V V (9코 소비, 15코 생성)
(1X, 1V^3) * 2    → X V^3 X V^3 (4코 소비, 8코 생성)
```

### 3.5 한 코 그룹 (Same-Hole Group)

대괄호 `[]`로 **한 부모 코에 여러 기호를 순서대로 적용**. `V`가 기본 늘림(같은 코에 `XX`)이라면, `[F,T]`는 한 코에 F와 T를 넣는다.

| 표기 | 의미 | 소비 코 → 생성 코 |
|---|---|---|
| `[F,T]` | 한 코에 F 1회 + T 1회 | 1 → 2 |
| `[F,F,F]` | 한 코에 F 3회 (V^3의 F 버전) | 1 → 3 |
| `[2F,T]` | 한 코에 F 2회 + T 1회 | 1 → 3 |
| `3[F,T]` | `[F,T]`를 3번 반복 | 3 → 6 |
| `[(F,T)*2]` | 한 코에 F,T를 2번 반복 | 1 → 4 |

**제약**:
- `[...]` 안에 **`V`, `A`는 사용 불가** — 이미 "한 코에 여러 개"를 뜻하는 기호라 중복/모호
- `[...]` **중첩 금지** (`[F,[X,T]]` 불가)
- `(...)` 는 **허용**

### 3.6 변형자 (Modifier)

`blo` 등은 **다음 기호에 적용**.

```
blo X             → 뒤이랑 짧은뜨기 1회
blo 6X            → 뒤이랑 짧은뜨기 6회
(blo X, V) * 3    → blo X 와 V를 교대로 3회
```

## 4. 단(Round/Row) 입력 형식

도안은 **단 단위로 입력**. 각 단은 한 줄로 작성.

```
1: @, 6X
2: 6V
3: (1X, 1V) * 6
4: (2X, 1V) * 6
5: (3X, 1V) * 6
6: 24X
```

- 단 번호 + `:` + 패턴
- 도형 종류는 별도 필드로 입력 (원형 / 평면)

## 5. MVP 범위

- **지원 도형**: 원형(circular), 평면(flat)
- **지원 기호**: `@`, `O`, `S`, `X`, `F`, `T`, `E`, `V`(+`^N`), `A`(+`^N`), `blo`
- **지원 그룹**: `()`(반복, `*N`), `[]`(한 코 그룹)
- **미지원(추후)**: 앞이랑(`flo`), 코 종류별 V/A 베이스 변경, 오각형/사각형 도형
